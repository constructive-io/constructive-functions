# Functions

This directory contains the playground functions for Constructive Functions.

## Structure

- `examples/` - Example functions demonstrating various patterns and capabilities

## Adding Functions

Functions in this directory can be deployed to the Kubernetes cluster using the configurations in the `k8s/` directory.

Each function should be self-contained and include:
- Function code
- Dependencies specification (package.json, requirements.txt, etc.)
- README with usage instructions

## Examples

See the `examples/` directory for sample functions.
